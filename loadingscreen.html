Interactive Loading Screen

An interactive loading screen where the goal is to entertain users while waiting for the assets to load with consistent frames not going below 40.

This game was made using SFML and for the partial fulfilment of our GDPARCM (Parallel Computing in Game Development). It served as our introduction towards Multithreading.
