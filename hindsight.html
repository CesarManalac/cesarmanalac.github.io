hindSIGHT

hindSIGHT is a short top-down shooter that implements the gun game rules of a First Person Shooter.

hindSIGHT was our submission for Global Game Jam 2023. It follows the theme of "Roots" where the theme lies in the guns and environment devolving as time goes on.

Role

For this project, I worked primarily as the UI Manager and Programmer focusing more on implementing core game mechanics and the functionality of the UI present.

My Contributions are as follows:

  • Level Design
  • Bug Testing
  • Audio Manager
  • Stopwatch Mechanic
